# UX 개선 & 추가 기능 설계 문서

## 1. 에디터 UX 개선 (12개)

### 1.1 블록 드래그 정교화
- 드래그 시 블록 고스트 이미지 표시 (반투명 복사본)
- 드롭 위치 인디케이터 (파란 수평선)
- 블록 사이, 블록 안(중첩)에 드롭 가능
- 드래그 시작 시 200ms 딜레이 (의도하지 않은 드래그 방지)

### 1.2 멀티 블록 복사/붙여넣기
- 블록 선택 상태에서 Cmd+C → 선택된 블록들 클립보드에 복사
- Cmd+V → 커서 위치에 블록 삽입
- 외부에서 복사한 HTML/텍스트도 블록으로 자동 변환

### 1.3 Undo/Redo 히스토리
- 비협업 모드: ProseMirror 기본 히스토리 (Cmd+Z / Cmd+Shift+Z)
- 협업 모드: Yjs UndoManager 연동 (이미 history: false 설정됨)
- Undo/Redo 버튼을 에디터 상단에 시각적으로 표시

### 1.4 블록 들여쓰기 Tab/Shift+Tab
- Tab: 현재 블록을 이전 블록의 자식으로 중첩
- Shift+Tab: 블록을 한 단계 위로 올림 (부모에서 분리)
- 리스트 항목에서는 리스트 레벨 변경
- 일반 블록에서는 column_list 구조로 변환하지 않고, 시각적 인덴트만 적용

### 1.5 포커스 모드
- 현재 편집 중인 블록만 하이라이트, 나머지 블록 반투명 처리
- Cmd+Shift+F로 토글
- 노션의 "Focus" 기능과 동일

### 1.6 전체 너비 토글 UX
- 페이지 상단 더보기(···) 메뉴에 "전체 너비" 토글 추가
- 토글 시 즉시 CSS max-width 변경 + DB 저장
- 아이콘으로 상태 표시

### 1.7 블록 핸들 개선
- 핸들이 항상 블록 왼쪽에 고정 (현재: 마우스 따라다님)
- 블록 호버 시 핸들 나타남 (opacity 0→1)
- 핸들 영역: + 버튼(블록 추가) + ⋮⋮ 버튼(드래그/메뉴)
- ⋮⋮ 버튼 위치가 블록 높이 중앙에 정렬

### 1.8 슬래시 커맨드 위치 개선
- 커서 바로 아래에 메뉴 표시 (현재: 가끔 화면 밖으로 벗어남)
- 화면 하단 근처일 때 위로 표시 (flip)
- 메뉴 최대 높이 제한 + 스크롤

### 1.9 인라인 링크 프리뷰 호버
- 링크 위에 마우스 호버 시 프리뷰 팝업
- URL, 페이지 제목(내부 링크), 외부 링크 OG 미리보기
- 300ms 딜레이 후 표시, 마우스 벗어나면 200ms 후 사라짐

### 1.10 마크다운 붙여넣기 자동 변환
- 클립보드에서 마크다운 텍스트 붙여넣기 시 자동으로 블록 변환
- `# 제목` → heading_1, `- 항목` → bulleted_list 등
- ProseMirror의 clipboardTextParser 활용

### 1.11 이미지 리사이즈 개선
- 현재: SE 코너만 리사이즈 가능
- 개선: 4모서리 + 4변 리사이즈 핸들
- 드래그 시 가이드라인 (25%, 50%, 75%, 100% 너비)
- 최소 크기 제한 (50px)

### 1.12 빈 블록 자동 정리
- 페이지 저장 시 연속 빈 paragraph 블록을 1개로 축소
- 마지막 블록은 항상 빈 paragraph 유지 (입력 영역)

## 2. 사이드바 & 네비게이션 (7개)

### 2.1 페이지 드래그앤드롭 순서/계층 변경
- 사이드바 페이지 아이템을 드래그하여 순서 변경
- 다른 페이지 위에 드롭하면 하위 페이지로 이동
- 드롭 인디케이터: 위/아래 수평선(순서), 들여쓰기(하위 페이지)
- `page.reorder` + `page.move` tRPC 호출

### 2.2 워크스페이스 스위처 드롭다운
- 사이드바 상단 워크스페이스 이름 클릭 시 드롭다운
- 소속된 워크스페이스 목록 표시
- 클릭하면 해당 워크스페이스로 전환 (URL 변경)
- "새 워크스페이스 만들기" 옵션

### 2.3 최근 방문 페이지 섹션
- 사이드바 상단에 "최근" 섹션 (즐겨찾기 위)
- 마지막 접근한 5개 페이지 표시
- 클라이언트 사이드 localStorage 기반 추적

### 2.4 사이드바 검색 필터
- 검색 결과에 필터 옵션: 생성자, 생성일, 수정일
- 커맨드 팔레트 내에 필터 칩 표시

### 2.5 페이지 아이콘 호버 프리뷰
- 사이드바 페이지 위에 마우스 호버 시 미리보기 툴팁
- 페이지 제목, 아이콘, 첫 몇 줄 콘텐츠 미리보기
- 300ms 딜레이

### 2.6 사이드바 접힘 상태에서 호버 확장
- 사이드바가 접혀있을 때 왼쪽 가장자리에 마우스 호버하면 임시 확장
- 마우스가 사이드바 밖으로 나가면 다시 접힘
- 노션과 동일한 동작

### 2.7 키보드 페이지 트리 탐색
- ↑↓: 페이지 아이템 간 이동 (포커스 링)
- Enter: 선택된 페이지 열기
- →: 하위 페이지 펼치기
- ←: 접기 또는 상위로 이동
- Home/End: 첫/마지막 페이지

## 3. 데이터베이스 UX (6개)

### 3.1 셀 인라인 편집 개선
- 셀 클릭 즉시 편집 모드 진입 (현재: 클릭 후 추가 클릭 필요)
- Tab: 다음 셀로 이동
- Shift+Tab: 이전 셀
- Enter: 다음 행
- Escape: 편집 취소

### 3.2 행 드래그 순서 변경
- 테이블 뷰에서 행 좌측 핸들 드래그로 순서 변경
- 보드 뷰에서 카드 드래그로 칼럼 간 이동 (기존) + 칼럼 내 순서 변경

### 3.3 그룹 접기/펼치기
- 테이블/리스트 뷰에서 그룹 헤더 클릭 시 접기/펼치기
- 접힌 그룹은 헤더만 표시 + 행 수 카운트
- 접기 상태 로컬 저장

### 3.4 뷰 탭 드래그 순서 변경
- 뷰 탭을 드래그하여 순서 변경
- `database.updateView` position 업데이트

### 3.5 속성 열 드래그 순서 변경
- 테이블 뷰에서 컬럼 헤더 드래그로 순서 변경
- `database.reorderProperties` 호출

### 3.6 데이터베이스 자동 행 높이
- 셀 내용이 긴 경우 행 높이 자동 확장
- 옵션: 고정 높이 / 자동 높이 토글

## 4. 협업 & 소통 UX (5개)

### 4.1 인라인 댓글 (텍스트 선택 → 댓글)
- 텍스트 선택 후 인라인 툴바에 💬 버튼
- 클릭하면 선택 범위에 하이라이트 + 댓글 입력 팝오버
- 해당 블록 + 텍스트 범위에 연결된 댓글 표시
- 하이라이트된 텍스트 호버 시 댓글 보기

### 4.2 @멘션 자동완성
- 에디터에서 `@` 입력 시 멘션 팝업
- 워크스페이스 멤버 목록에서 검색/선택
- `@페이지이름` → 페이지 링크
- `@날짜` → 날짜 선택기
- 멘션 시 대상 사용자에게 알림 생성

### 4.3 실시간 타이핑 인디케이터
- 다른 사용자가 타이핑 중일 때 "OOO님이 입력 중..." 표시
- 에디터 하단 또는 탑바에 표시
- Yjs awareness 활용

### 4.4 읽은 위치 동기화
- 다른 사용자의 현재 스크롤 위치를 탑바 아바타 옆에 표시
- 아바타 클릭 시 해당 위치로 스크롤

### 4.5 댓글 이모지 리액션
- 댓글에 이모지 리액션 버튼 (👍 ❤️ 😊 🎉 😮 😢)
- 리액션 카운트 표시
- DB: Comment에 reactions JSON 필드 추가

## 5. 비주얼 폴리시 & 모션 (4개)

### 5.1 페이지 전환 애니메이션
- 페이지 이동 시 콘텐츠 fade-in (opacity 0→1, 150ms)
- 사이드바에서 페이지 선택 시 메인 콘텐츠 부드러운 전환
- Next.js App Router의 `loading.tsx` 활용

### 5.2 사이드바 모션 개선
- 접기/펼치기 시 easeInOut 커브 (현재 linear에 가까움)
- 페이지 트리 확장/축소 시 height 애니메이션
- 드래그 리사이즈 시 실시간 반응 (현재도 동작하지만 부드러움 개선)

### 5.3 토스트 애니메이션 개선
- 입장: slide-up + fade-in (현재: translate-y + opacity)
- 퇴장: slide-down + fade-out
- 스택: 여러 토스트 시 아래에서 위로 쌓임 + 위 토스트들 자동 이동

### 5.4 모달 트랜지션
- 배경: opacity 0→0.6 (150ms)
- 모달: scale 0.95→1 + opacity 0→1 (200ms)
- 닫힘: 역순 (150ms)
- 모든 모달(설정, 공유, 휴지통 등)에 일관 적용

## 6. 편의 기능 (8개)

### 6.1 페이지 템플릿 시스템
- 새 페이지 생성 시 템플릿 선택 화면
- 기본 템플릿: 빈 페이지, 회의록, 할 일 목록, 주간 계획, 프로젝트 계획
- 커스텀 템플릿: 현재 페이지를 템플릿으로 저장
- Page.isTemplate 필드 활용 (이미 스키마에 존재)

### 6.2 키보드 단축키 도움말
- Cmd+/ 누르면 모든 단축키 목록 모달
- 카테고리별 그룹: 에디터, 네비게이션, 서식
- 검색 가능

### 6.3 빠른 액션 (Cmd+Shift+P)
- VS Code 스타일 명령 팔레트
- 기존 Cmd+K (검색) + 명령 실행 기능 확장
- "페이지 잠금", "전체 너비", "내보내기", "공유" 등 빠른 실행

### 6.4 페이지 히스토리 & 복원
- 페이지 편집 히스토리 (스냅샷 기반)
- 특정 시점으로 복원
- PageSnapshot 모델 추가
- 자동 스냅샷: 30분마다 또는 중요 변경 시

### 6.5 마크다운 가져오기
- .md 파일 드래그앤드롭 또는 파일 선택
- 마크다운 파싱 → 블록 변환
- 제목, 리스트, 코드, 이미지 URL, 링크 지원

### 6.6 PDF 내보내기
- 페이지를 PDF로 다운로드
- 서버 사이드: puppeteer 또는 react-pdf
- 헤더/푸터 옵션 (페이지 제목, 날짜)

### 6.7 워크스페이스 멤버 온라인 상태
- 설정 > 멤버 목록에서 온라인/오프라인 표시
- 녹색 점: 현재 접속 중
- Yjs awareness 또는 WebSocket 기반

### 6.8 다크모드 자동 전환 개선
- 시스템 설정 변경 시 자동 전환 (현재: 수동 새로고침 필요)
- `matchMedia("prefers-color-scheme")` change 이벤트 리스너
- 전환 시 부드러운 트랜지션 (300ms)

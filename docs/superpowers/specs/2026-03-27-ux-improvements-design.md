# UX 개선 & 추가 기능 종합 설계 문서

## 1. 에디터 코어 UX (22개)

### 1.1 블록 드래그 정교화
- 드래그 시 블록 고스트 이미지 (반투명 복사본, 실제 블록 크기의 80%)
- 드롭 위치 인디케이터: 블록 사이 → 파란 수평선 2px, 블록 안 → 파란 테두리 (중첩)
- 드래그 중 자동 스크롤 (상단/하단 50px 영역에 진입 시)
- 드래그 시작 200ms 딜레이 (미세 클릭과 구분)
- Cmd+드래그: 블록 복사 (이동 대신)
- 블록 간 드래그뿐 아니라 칼럼 간 드래그도 지원

### 1.2 멀티 블록 복사/붙여넣기/잘라내기
- 블록 선택 상태에서 Cmd+C → 선택된 블록들 내부 포맷으로 클립보드 복사
- Cmd+X → 잘라내기 (복사 후 삭제)
- Cmd+V → 커서 위치에 블록 삽입
- 블록 선택 후 Cmd+D → 선택된 블록 바로 아래에 복제
- 다른 페이지에서 복사한 블록도 붙여넣기 가능
- 외부 HTML 붙여넣기 시 자동 블록 변환 (Google Docs, Word 등)

### 1.3 Undo/Redo
- 비협업: ProseMirror History (Cmd+Z / Cmd+Shift+Z)
- 협업: Yjs UndoManager 자동 연동 (history: false 이미 설정)
- 토스트에 "실행 취소" 링크 표시 (삭제, 이동 등 주요 액션)
- Undo 스택 최대 100단계

### 1.4 블록 들여쓰기 Tab/Shift+Tab
- Tab: 리스트 → 하위 레벨로 들여쓰기, 일반 블록 → 이전 블록의 자식으로 중첩
- Shift+Tab: 역방향 (상위 레벨로)
- 멀티 블록 선택 상태에서 Tab → 선택된 블록 모두 들여쓰기
- 코드 블록 내에서 Tab → 실제 탭 문자 삽입 (들여쓰기가 아님)

### 1.5 포커스 모드
- Cmd+Shift+F로 토글
- 활성화 시: 현재 편집 블록은 opacity 1, 나머지 블록 opacity 0.15
- 블록 이동 시 자연스러운 opacity 전환 (200ms)
- 사이드바/탑바도 반투명 처리

### 1.6 블록 핸들 고정 개선
- 핸들 위치: 블록 왼쪽 -40px에 고정 (현재: 마우스 따라다님)
- 블록 호버 시 150ms 후 나타남, 블록 벗어나면 200ms 후 사라짐
- + 버튼: 블록 추가 (해당 위치에 빈 paragraph 삽입)
- ⋮⋮ 버튼: 클릭 → 블록 선택(파란 하이라이트), 드래그 → 블록 이동
- 핸들 높이: 블록 첫 줄 기준으로 세로 중앙 정렬
- 중첩 블록(리스트 아이템 등)에도 각각 핸들 표시

### 1.7 슬래시 커맨드 위치 & 스크롤
- 커서 바로 아래에 메뉴 표시
- 화면 하단 200px 이내면 위로 flip
- 화면 우측 벗어남 방지 (좌측으로 이동)
- 메뉴 내 키보드 탐색 시 선택된 항목이 보이도록 자동 스크롤
- 카테고리 사이 구분선 + 카테고리 이름 고정 헤더

### 1.8 인라인 링크 프리뷰
- 링크 호버 300ms 후 프리뷰 팝업 표시
- 내부 링크: 페이지 아이콘 + 제목 + 첫 2줄 텍스트
- 외부 링크: OG 이미지 + 제목 + 설명 + URL
- 프리뷰 팝업에서 "열기", "복사", "삭제" 버튼
- 마우스가 팝업 위에 있으면 유지, 벗어나면 200ms 후 사라짐

### 1.9 마크다운 붙여넣기 자동 변환
- 마크다운 텍스트 붙여넣기 시 블록 자동 변환
- `# ` → H1, `## ` → H2, `### ` → H3
- `- ` → 불릿 리스트, `1. ` → 번호 리스트, `- [ ] ` → 체크리스트
- `> ` → 인용, ` ``` ` → 코드 블록
- `---` → 구분선
- `**bold**` → 볼드, `*italic*` → 이탤릭, `` `code` `` → 인라인 코드
- `[text](url)` → 링크, `![alt](url)` → 이미지

### 1.10 이미지 리사이즈 개선
- 8개 리사이즈 핸들 (4코너 + 4변)
- 드래그 시 너비 가이드라인 표시 (25%, 50%, 75%, 100%)
- 리사이즈 중 실시간 크기 표시 (px 또는 %)
- Shift+드래그: 비율 유지 리사이즈
- 최소 50px, 최대 페이지 너비
- 더블클릭: 원본 크기 복원

### 1.11 빈 블록 자동 정리
- 저장 시 연속 빈 paragraph 2개 이상 → 1개로 축소
- 마지막 블록은 항상 빈 paragraph 유지
- 이미지/임베드 뒤에 자동으로 빈 paragraph 추가 (입력 영역 확보)

### 1.12 블록 우클릭 컨텍스트 메뉴
- 블록 위에서 우클릭 → 노션 스타일 컨텍스트 메뉴
- 메뉴: 삭제, 복제, 블록 타입 변환, 색상, 링크 복사, 페이지로 변환, 위로 이동, 아래로 이동
- 인라인 툴바 대신 우클릭으로도 접근 가능

### 1.13 블록 → 페이지 변환 ("Turn into page")
- 블록 메뉴 또는 우클릭에서 "페이지로 변환" 옵션
- 선택한 블록의 텍스트를 제목으로, 하위 블록들을 새 페이지 콘텐츠로
- 원래 위치에는 새 페이지 링크(link_to_page) 블록 삽입

### 1.14 텍스트 정렬 (Left / Center / Right)
- 인라인 툴바에 정렬 버튼 추가
- 블록 단위로 text-align 설정
- 기본: left, 선택: center, right
- 이미지 블록도 정렬 지원

### 1.15 코드 블록 개선
- 언어 자동 감지 (첫 몇 줄 기반 heuristic)
- "복사" 버튼 (코드 블록 우상단)
- 줄 번호 표시 토글
- 코드 블록 내 Tab → 2스페이스 들여쓰기
- 언어 선택 드롭다운 검색 가능

### 1.16 토글 헤딩 (Toggle Heading)
- H1/H2/H3가 토글로도 동작 (노션의 "Toggle heading")
- 슬래시 커맨드: "토글 제목 1", "토글 제목 2", "토글 제목 3"
- 헤딩 좌측에 ▶ 아이콘, 클릭하면 하위 콘텐츠 접기/펼치기

### 1.17 심플 테이블 행/열 추가 삭제
- 테이블 호버 시 우측에 + (열 추가), 하단에 + (행 추가)
- 행/열 헤더 우클릭 → 삭제, 위/아래 추가, 좌/우 추가
- 행/열 드래그로 순서 변경
- 셀 병합/분할

### 1.18 수식 실시간 미리보기
- 수식 블록 편집 시 LaTeX 입력 아래에 KaTeX 렌더링 실시간 표시
- 인라인 수식도 동일 (입력 중 옆에 미리보기)
- 에러 시 빨간 텍스트로 에러 메시지

### 1.19 Synced Block (동기화 블록)
- 블록을 "동기화 블록"으로 변환 → 다른 페이지에서 참조 가능
- 원본 변경 시 모든 참조 위치에서 동기화
- 빨간 테두리로 동기화 블록 표시 (노션 스타일)
- SyncedBlock 모델 추가 (sourceBlockId, references)

### 1.20 Link to Page 블록
- 다른 페이지에 대한 링크 블록
- 슬래시 커맨드 "페이지 링크" → 페이지 검색/선택 팝업
- 아이콘 + 제목 형태로 렌더링
- 클릭하면 해당 페이지로 이동

### 1.21 클립보드 이미지 붙여넣기
- Cmd+V로 클립보드 이미지 직접 붙여넣기
- 스크린샷, 복사한 이미지 자동 업로드
- 업로드 중 로딩 표시

### 1.22 파일 드래그앤드롭 에디터로
- 데스크탑에서 파일을 에디터 영역으로 드래그앤드롭
- 이미지 → 이미지 블록, 동영상 → 동영상 블록, 기타 → 파일 블록
- 드래그 중 에디터에 드롭존 오버레이 표시

## 2. 사이드바 & 네비게이션 (12개)

### 2.1 페이지 드래그앤드롭
- 드래그로 순서 변경 (같은 레벨)
- 다른 페이지 위에 드롭: 하위 페이지로 이동 (300ms 홀드 후 확장)
- 드롭 인디케이터: 수평선(순서), 들여쓰기(하위)
- 즐겨찾기 섹션도 드래그로 순서 변경 가능
- `page.reorder` + `page.move` API 호출

### 2.2 워크스페이스 스위처
- 상단 워크스페이스 이름 클릭 → 드롭다운 메뉴
- 소속 워크스페이스 목록 (아이콘 + 이름 + 멤버 수)
- "새 워크스페이스" 생성 옵션
- 활성 워크스페이스 체크마크 표시
- 로그아웃 버튼

### 2.3 최근 방문 페이지
- 사이드바 상단 "최근" 섹션 (접기 가능)
- 마지막 접근 5개 페이지 (localStorage 기반)
- 아이콘 + 제목, 호버 시 전체 경로 툴팁

### 2.4 사이드바 검색 필터
- 커맨드 팔레트(Cmd+K) 내 필터 칩
- 필터: 생성자, 생성일 범위, 수정일 범위, 페이지 타입(일반/DB)
- 정렬: 관련도순, 최신순, 이름순

### 2.5 페이지 호버 프리뷰 (Peek)
- 사이드바 페이지 위 500ms 호버 → 프리뷰 카드
- 카드: 커버 이미지(있으면), 아이콘 + 제목, 첫 3줄 텍스트, 마지막 편집일
- 카드 위치: 사이드바 오른쪽에 표시

### 2.6 사이드바 접힘 시 호버 확장
- 접힌 상태에서 좌측 가장자리 호버 → 사이드바 오버레이로 임시 확장
- 메인 콘텐츠는 밀리지 않음 (오버레이)
- 마우스 벗어나면 500ms 후 다시 접힘
- 클릭하면 완전히 펼침 (고정)

### 2.7 키보드 페이지 트리 탐색
- ↑↓: 아이템 간 이동 (포커스 링 표시)
- Enter: 페이지 열기
- →: 펼치기, ←: 접기 또는 상위 이동
- Cmd+↑: 페이지를 위로 이동 (순서 변경)
- Cmd+↓: 아래로 이동
- Delete: 휴지통으로 이동 (확인 토스트)
- F2: 페이지 이름 인라인 편집

### 2.8 브레드크럼 클릭 → 자식 드롭다운
- 브레드크럼의 각 아이템 클릭 시 해당 페이지의 하위 페이지 드롭다운
- 하위 페이지 빠른 탐색
- 드롭다운에서 "새 하위 페이지" 생성 옵션

### 2.9 페이지 이동 다이얼로그
- 블록 메뉴 또는 더보기 메뉴에서 "이동" 옵션
- 워크스페이스 내 페이지 트리를 보여주는 다이얼로그
- 대상 페이지 선택 → 하위 페이지로 이동
- 검색으로 빠른 대상 페이지 찾기

### 2.10 사이드바 섹션 접기
- "즐겨찾기", "개인 페이지" 등 섹션 헤더 클릭 시 접기/펼치기
- 접기 상태 localStorage 저장
- 접힌 섹션 옆에 페이지 수 배지

### 2.11 페이지 아이콘 인라인 변경
- 사이드바 페이지 아이콘 클릭 → 이모지 피커 팝업 (페이지 뷰에 가지 않고)
- 빠른 아이콘 변경

### 2.12 탑바 더보기(···) 메뉴 확장
- 현재 페이지 관련 모든 액션을 하나의 더보기 메뉴에:
  - 페이지 잠금/해제
  - 전체 너비 토글
  - 즐겨찾기 추가/제거
  - 복제
  - 이동
  - 내보내기 (MD/HTML/PDF)
  - 삭제 (휴지통)
  - 히스토리 보기
  - 단어 수 표시
  - 마지막 편집 시간/편집자

## 3. 데이터베이스 UX (14개)

### 3.1 셀 인라인 편집 개선
- 싱글 클릭 → 즉시 편집 모드
- Tab → 다음 셀, Shift+Tab → 이전 셀
- Enter → 다음 행 같은 열, Shift+Enter → 셀 내 줄바꿈
- Escape → 편집 취소
- 텍스트 셀: 클릭 즉시 커서 배치
- Select 셀: 클릭 즉시 드롭다운 열림

### 3.2 행 드래그 순서 변경
- 테이블: 행 좌측 핸들 드래그
- 보드: 카드 칼럼 내 순서 + 칼럼 간 이동
- 리스트: 행 드래그 순서
- 드롭 인디케이터 표시

### 3.3 그룹 접기/펼치기
- 그룹 헤더 클릭 → 접기/펼치기
- 접힌 상태: 헤더 + 행 수 카운트만 표시
- "모두 접기" / "모두 펼치기" 버튼
- 접기 상태 로컬 저장

### 3.4 뷰 탭 드래그 순서
- 뷰 탭 드래그로 순서 변경
- 뷰 탭 우클릭 → 복제, 삭제, 이름 변경

### 3.5 속성 열 드래그 순서
- 테이블 뷰 컬럼 헤더 드래그로 순서 변경
- 드롭 인디케이터 (파란 수직선)

### 3.6 행 높이 자동/고정 토글
- 뷰 설정에서 "행 높이" 옵션: 기본(1줄), 자동(내용에 맞춤)
- 자동 높이 시 긴 텍스트가 여러 줄로 표시

### 3.7 데이터베이스 행 열기 (피크 뷰)
- 행 클릭 → 오른쪽 사이드 피크 패널로 열기 (현재: 새 페이지로 이동)
- 피크 패널에서 속성 편집 + 블록 에디터
- 전체 페이지로 열기 버튼 제공
- ESC로 닫기

### 3.8 캘린더 드래그로 날짜 변경
- 캘린더 뷰에서 카드를 다른 날짜로 드래그 → date 속성 자동 변경
- 드래그 중 날짜 하이라이트

### 3.9 타임라인 드래그로 기간 변경
- 타임라인 뷰에서 바 양쪽 드래그로 시작/종료 날짜 변경
- 바 전체 드래그로 날짜 이동 (기간 유지)

### 3.10 링크드 데이터베이스 (다른 DB의 뷰)
- 기존 데이터베이스를 다른 페이지에서 참조하는 "링크드 뷰"
- 원본 DB의 데이터를 다른 필터/정렬/뷰로 표시

## 5. 비주얼 폴리시 & 모션 (10개)

### 5.1 페이지 전환 애니메이션
- 콘텐츠 fade-in (opacity 0→1, 150ms ease-out)
- 콘텐츠 약간 위로 슬라이드 (translateY 8px→0)
- loading.tsx 활용한 스켈레톤 로딩 상태

### 5.2 사이드바 모션
- 접기/펼치기: cubic-bezier(0.4, 0, 0.2, 1), 300ms
- 페이지 트리 노드 확장: height auto 애니메이션 (max-height 트릭)
- 드래그 리사이즈: transform으로 실시간 반응 (layout 트리거 방지)

### 5.3 토스트 스택 애니메이션
- 입장: translateY(100%) → translateY(0) + opacity 0→1, 300ms spring
- 퇴장: translateY(0) → translateY(100%) + opacity 1→0, 200ms ease-in
- 스택: 새 토스트 추가 시 기존 토스트들 위로 이동 (40px gap)
- 최대 3개 동시 표시, 초과 시 가장 오래된 것 자동 제거

### 5.4 모달/다이얼로그 트랜지션
- 배경: opacity 0→0.6, 200ms
- 모달: scale(0.96)→scale(1) + opacity(0→1), 200ms cubic-bezier
- 닫힘: 역순, 150ms
- 모든 모달 컴포넌트에 일관 적용 (공유, 설정, 휴지통, 이모지 피커 등)

### 5.5 드롭다운/팝오버 애니메이션
- 열림: scaleY(0.95)→scaleY(1) + opacity, origin top, 150ms
- 닫힘: 즉시 (사용자 의도 존중)
- 슬래시 메뉴, 블록 메뉴, 컬러 피커, AI 메뉴 등 모든 드롭다운에 적용

### 5.6 블록 삽입/삭제 애니메이션
- 새 블록 삽입: height 0→auto + opacity 0→1, 200ms
- 블록 삭제: height auto→0 + opacity 1→0, 150ms
- 블록 이동: 부드러운 위치 전환 (FLIP 애니메이션)

### 5.7 셀렉션 애니메이션
- 블록 선택 하이라이트: 부드러운 배경색 전환 (150ms)
- 텍스트 선택 시 인라인 툴바 부드럽게 나타남 (opacity + translateY)

### 5.8 로딩 스켈레톤
- 페이지 로딩 중 스켈레톤 UI (노션 스타일 펄스 애니메이션)
- 사이드바 페이지 트리 로딩 스켈레톤
- 데이터베이스 로딩 스켈레톤

### 5.9 에러 바운더리 UI
- 컴포넌트 에러 시 크래시 대신 에러 메시지 + "다시 시도" 버튼
- 에디터 에러, 사이드바 에러, 데이터베이스 에러 각각 분리

### 5.10 스크롤 관련
- 부드러운 스크롤 (scroll-behavior: smooth)
- 페이지 상단으로 스크롤 버튼 (우하단, 스크롤 500px 이후 표시)
- 목차(TOC) 클릭 시 해당 헤딩으로 부드러운 스크롤

## 1. 에디터 코어 UX (28개)

### 1.1 블록 드래그 정교화
- **고스트 이미지**: 드래그 중 반투명(0.8) 블록 복사본이 마우스를 따라다님, 원본 크기의 80%
- **드롭 인디케이터**: 블록 사이 → 2px 파란 수평선, 블록 안(중첩) → 파란 테두리(2px)
- **자동 스크롤**: 드래그 중 상단/하단 50px 영역 진입 시 초당 200px 스크롤
- **드래그 딜레이**: 마우스다운 후 200ms 이상 이동해야 드래그 시작 (미세 클릭 방지)
- **Cmd+드래그**: 이동 대신 복사 (블록 복제 후 드롭 위치에 삽입)
- **칼럼 간 드래그**: column_list 내 칼럼 사이로 블록 이동 가능
- **드래그 취소**: ESC로 드래그 취소, 원래 위치로 복원

### 1.2 멀티 블록 복사/잘라내기/붙여넣기/복제
- **Cmd+C**: 선택된 블록들을 내부 포맷(JSON) + HTML 양쪽으로 클립보드 복사
- **Cmd+X**: 잘라내기 (복사 후 삭제)
- **Cmd+V**: 커서 위치에 블록 삽입. 노션 내부 포맷이면 블록으로, HTML이면 자동 변환, 텍스트면 paragraph로
- **Cmd+D**: 선택된 블록 바로 아래에 복제 (빠른 복제)
- **다른 페이지에서 복사한 블록도 붙여넣기 가능** (세션 내 클립보드 공유)
- **외부 HTML 변환**: Google Docs, Word, Excel에서 복사한 내용 → 적절한 블록으로 자동 변환
  - 표 → 테이블 블록, 리스트 → 리스트 블록, 볼드/이탤릭 → 마크 유지

### 1.3 Undo/Redo 개선
- **비협업**: ProseMirror History (Cmd+Z / Cmd+Shift+Z), 최대 100단계
- **협업**: Yjs UndoManager 연동 (현재 history:false 설정되어 있음, UndoManager 바인딩 필요)
- **토스트 Undo 링크**: 삭제, 이동, 블록 타입 변환 등 주요 액션 시 토스트에 "실행 취소" 클릭 가능 링크
- **Undo 스택 시각화**: 없음 (노션과 동일 — UI 없이 Cmd+Z만)

### 1.4 블록 들여쓰기 Tab/Shift+Tab
- **Tab**: 리스트 → 하위 레벨 들여쓰기. 일반 블록 → 이전 블록의 자식으로 중첩
- **Shift+Tab**: 역방향 (상위 레벨)
- **멀티 블록 선택 + Tab**: 선택된 블록 모두 한 단계 들여쓰기
- **코드 블록 내**: Tab → 2스페이스 삽입 (들여쓰기가 아님)
- **테이블 셀 내**: Tab → 다음 셀로 이동 (들여쓰기 아님)

### 1.5 포커스 모드
- **Cmd+Shift+F** 토글
- 활성 블록: opacity 1, 나머지: opacity 0.15
- 블록 이동 시 200ms ease 전환
- 사이드바/탑바도 opacity 0.3으로 반투명
- ESC로 해제 가능

### 1.6 블록 핸들 고정 & 개선
- **위치**: 블록 왼쪽 -44px에 고정 (현재: 마우스 위치에 따라다님)
- **나타남/사라짐**: 블록 호버 150ms 후 나타남, 벗어나면 200ms 후 사라짐
- **+ 버튼**: 해당 위치 아래에 빈 paragraph 삽입
- **⋮⋮ 버튼**: 클릭 → 블록 선택(파란 하이라이트) + 메뉴 열기. 드래그 → 블록 이동
- **높이 정렬**: 블록 첫 줄 기준 세로 중앙
- **중첩 블록**: 리스트 아이템, 토글 내부 블록에도 각각 핸들 표시
- **선택된 블록**: 핸들 항상 표시 (호버 불필요)

### 1.7 슬래시 커맨드 위치 & UX
- **위치**: 커서 바로 아래에 표시
- **화면 하단 200px 이내**: 위로 flip (메뉴가 커서 위에)
- **화면 우측 벗어남**: 좌측으로 이동
- **키보드 탐색**: 선택 항목이 뷰포트 밖이면 자동 스크롤
- **카테고리 헤더**: 스크롤 시 상단에 고정 (sticky)
- **최근 사용**: 최근 사용한 블록 타입 3개를 상단에 "최근 사용" 섹션으로 표시
- **빈 검색 결과**: "결과 없음" 메시지 + "텍스트로 입력" 폴백

### 1.8 인라인 링크 프리뷰
- **호버 300ms 후** 프리뷰 팝업 표시
- **내부 링크** (페이지): 아이콘 + 제목 + 첫 2줄 + 마지막 편집 시간
- **외부 링크**: OG 이미지 + 제목 + 설명 + URL 도메인
- **팝업 액션**: "열기" (새 탭), "링크 복사", "링크 제거"
- **팝업 유지**: 마우스가 팝업 위에 있으면 유지, 벗어나면 200ms 후 사라짐
- **편집 모드**: 링크 클릭 시 URL 편집 가능 (인라인 입력)

### 1.9 마크다운 붙여넣기 자동 변환
- 마크다운 텍스트 붙여넣기 → 블록 자동 변환:
  - `# ` → H1, `## ` → H2, `### ` → H3
  - `- ` / `* ` → 불릿 리스트, `1. ` → 번호 리스트
  - `- [ ] ` / `- [x] ` → 체크리스트 (체크 상태 유지)
  - `> ` → 인용, ` ``` ` → 코드 블록 (언어 감지)
  - `---` / `***` → 구분선
  - `**bold**` → 볼드, `*italic*` → 이탤릭, `` `code` `` → 인라인 코드, `~~strike~~` → 취소선
  - `[text](url)` → 링크, `![alt](url)` → 이미지 블록
  - `| a | b |` → 테이블 블록
- **ProseMirror clipboardTextParser** + 커스텀 마크다운 파서로 구현
- **붙여넣기 후 Cmd+Z**: 원래 텍스트로 복원 (변환 실행 취소)

### 1.10 이미지 리사이즈 개선
- **8개 핸들**: 4코너(nwse, nesw) + 4변(ew, ns)
- **가이드라인**: 드래그 중 점선 표시 (25%, 50%, 75%, 100% 페이지 너비)
- **실시간 크기 표시**: 드래그 중 우하단에 "320 × 240" 또는 "50%" 뱃지
- **Shift+드래그**: 비율 고정 리사이즈
- **최소/최대**: 최소 50px, 최대 페이지 너비 100%
- **더블클릭**: 원본 크기 복원
- **정렬**: 리사이즈 후에도 정렬(left/center/right) 유지

### 1.11 빈 블록 자동 정리
- 저장 시 연속 빈 paragraph 2개 이상 → 1개로 축소
- 마지막 블록은 항상 빈 paragraph 유지 (입력 영역)
- 이미지/임베드/파일 블록 뒤에 자동으로 빈 paragraph 추가

### 1.12 블록 우클릭 컨텍스트 메뉴
- 에디터 내 우클릭 → 커스텀 컨텍스트 메뉴 (브라우저 기본 메뉴 대체)
- 메뉴 항목:
  - **삭제** (Backspace 아이콘)
  - **복제** (Cmd+D)
  - **블록 타입 변환** → 서브메뉴: 텍스트, H1, H2, H3, 불릿, 번호, 체크, 인용, 코드
  - **색상** → 서브메뉴: 텍스트 10색 + 배경 10색
  - **링크 복사** (블록 앵커 URL)
  - **페이지로 변환** (1.13)
  - **위로 이동** / **아래로 이동**
  - **AI에게 요청** (SP-9 연동)
  - **댓글 추가** (4.1 연동)
- 텍스트 선택 상태에서 우클릭: 인라인 서식 옵션도 포함

### 1.13 블록 → 페이지 변환 ("Turn into page")
- 블록 메뉴/우클릭에서 "페이지로 변환"
- 블록 텍스트 → 새 페이지 제목
- 하위 블록들 → 새 페이지 콘텐츠로 이동
- 원래 위치에 link_to_page 블록 삽입 (아이콘 + 제목)
- 역방향: 하위 페이지를 "인라인으로 변환" → 부모 페이지에 블록으로 삽입

### 1.14 텍스트 정렬 (Left / Center / Right)
- 인라인 툴바에 정렬 드롭다운 추가 (⫶ 아이콘)
- 블록 단위 text-align: left(기본), center, right
- 이미지 블록도 정렬 지원 (float left/center/right)
- 데이터 저장: 블록 attrs에 `textAlign` 필드

### 1.15 코드 블록 개선
- **언어 자동 감지**: 첫 5줄 기반 heuristic (주요 언어 20종)
- **복사 버튼**: 코드 블록 우상단에 📋 버튼, 클릭 시 코드 전체 복사 + "복사됨" 토스트
- **줄 번호**: 토글 가능 (기본: off), 코드 블록 설정 메뉴에서 on/off
- **Tab → 2스페이스**: 코드 블록 내 Tab 키 동작
- **Shift+Tab → 2스페이스 제거**: 들여쓰기 해제
- **언어 드롭다운**: 검색 가능, 인기 언어 상단 고정 (JavaScript, TypeScript, Python, Go, Rust, SQL, JSON, YAML, Bash, HTML, CSS)
- **코드 접기**: 긴 코드 블록(20줄 이상) 접기/펼치기 토글

### 1.16 토글 헤딩
- **토글 제목 1/2/3**: H1/H2/H3 + 토글 기능 결합
- 슬래시 커맨드: "토글 제목 1", "토글 제목 2", "토글 제목 3"
- 헤딩 좌측에 ▶ 아이콘, 클릭 시 하위 콘텐츠 접기/펼치기
- 접힌 상태에서 헤딩 텍스트 옆에 "..." 표시 (콘텐츠 있음을 암시)
- Tiptap: heading 확장에 `isToggle: boolean` 어트리뷰트 추가

### 1.17 심플 테이블 조작 개선
- **행/열 추가**: 테이블 호버 시 우측에 +열, 하단에 +행 버튼
- **행/열 삭제**: 행/열 헤더 우클릭 → 삭제
- **행/열 삽입**: 우클릭 → "위에 행 추가", "아래에 행 추가", "왼쪽에 열 추가", "오른쪽에 열 추가"
- **행/열 드래그 순서 변경**: 헤더 영역 드래그
- **셀 병합/분할**: 셀 다중 선택 → 우클릭 → "셀 병합"
- **헤더 행 토글**: 첫 행을 헤더로 지정/해제
- **테이블 전체 너비 토글**: 페이지 너비에 맞춤 vs 콘텐츠에 맞춤

### 1.18 수식 실시간 미리보기
- 수식 블록 편집 시: 입력 필드 아래에 KaTeX 렌더링 실시간 표시
- 인라인 수식: 입력 중 옆에 미리보기 팝업
- 에러 시: 빨간 텍스트로 에러 메시지 + 에러 위치 하이라이트
- 수식 자동완성: `\alpha`, `\beta`, `\sum`, `\frac` 등 자주 쓰는 명령어

### 1.19 동기화 블록 (Synced Block)
- 블록 메뉴에서 "동기화 블록으로 변환"
- 다른 페이지에서 "동기화 블록 삽입" (원본 블록 검색)
- 원본 수정 → 모든 참조 위치에서 실시간 반영
- 빨간 테두리 + "동기화됨" 라벨
- DB 모델: `SyncedBlock { id, sourceBlockId, pageId, position }`

### 1.20 Link to Page 블록
- 슬래시 커맨드 "페이지 링크" → 페이지 검색 팝업 (Cmd+K와 유사)
- 선택하면 아이콘 + 제목 형태 블록 삽입
- 클릭 → 해당 페이지로 이동
- 대상 페이지 삭제 시 "삭제된 페이지" 스타일로 표시

### 1.21 클립보드 이미지 붙여넣기
- Cmd+V로 클립보드 이미지 직접 에디터에 삽입
- 스크린샷, 브라우저에서 복사한 이미지 감지
- 자동 업로드 (/api/upload) + 이미지 블록 삽입
- 업로드 중 로딩 스피너 표시, 실패 시 에러 토스트

### 1.22 파일 드래그앤드롭 → 에디터
- 데스크탑 파일을 에디터 영역으로 드래그
- **드롭존 오버레이**: 에디터 전체에 파란 반투명 오버레이 + "파일을 놓으세요" 텍스트
- 파일 타입별 자동 블록 생성:
  - 이미지 (jpg, png, gif, webp, svg) → 이미지 블록
  - 동영상 (mp4, webm, mov) → 동영상 블록
  - 오디오 (mp3, wav, ogg) → 오디오 블록
  - 기타 → 파일 블록
- 여러 파일 동시 드롭 → 순서대로 블록 생성
- 업로드 진행 표시 (각 파일별)

### 1.23 블록 색상 키보드 단축키
- 블록 선택 상태에서 `/color` 또는 Cmd+Shift+H → 색상 팔레트 팝업
- 숫자 키로 빠른 선택 (1:빨강, 2:주황, 3:노랑, ... 0:기본)
- 텍스트 선택 → 같은 단축키로 텍스트 색상

### 1.24 블록 이동 단축키
- **Cmd+Shift+↑**: 현재 블록을 위로 한 칸 이동
- **Cmd+Shift+↓**: 현재 블록을 아래로 한 칸 이동
- 멀티 블록 선택 상태에서도 동작

### 1.25 블록 타입 변환 단축키
- 블록 시작에서 타이핑: `/h1`, `/h2`, `/h3`, `/quote`, `/code`, `/todo`, `/bullet`, `/number`
- 기존 마크다운 단축키 외 추가 변환 지원

### 1.26 전체 너비 토글
- 탑바 더보기 메뉴에서 "전체 너비" 토글
- 또는 Cmd+Shift+W 단축키
- 토글 시 부드러운 max-width 전환 애니메이션 (300ms)
- DB에 즉시 저장 (page.isFullWidth)

### 1.27 텍스트 하이라이트 (형광펜)
- 인라인 툴바에 형광펜 버튼
- 텍스트 배경색 10가지 (노션 배경색과 동일)
- Cmd+Shift+H로 빠른 토글 (마지막 사용 색상)
- Tiptap Highlight 확장 활용 (이미 설치됨, UI만 연결)

### 1.28 찾기 & 바꾸기 (Cmd+F / Cmd+H)
- **Cmd+F**: 페이지 내 텍스트 검색 바 (에디터 상단)
  - 검색어 하이라이트 (노란색 배경)
  - ↑↓ 탐색 (이전/다음 결과)
  - 매칭 수 표시 ("3/15")
  - 대소문자 구분 토글
- **Cmd+H**: 바꾸기 모드
  - "바꾸기" + "모두 바꾸기" 버튼

## 2. 사이드바 & 네비게이션 (15개)

### 2.1 페이지 드래그앤드롭
- 사이드바 페이지 아이템 드래그로 순서 변경 (같은 레벨)
- 다른 페이지 위에 300ms 홀드 → 자동 확장 + 하위 페이지로 드롭 가능
- **드롭 인디케이터**: 위/아래 수평선(순서), 들여쓰기(하위 페이지)
- 즐겨찾기 섹션도 드래그 순서 변경
- 다른 섹션으로 이동 불가 (즐겨찾기 → 개인 페이지 X)
- `page.reorder` + `page.move` API 호출
- 드래그 취소: ESC

### 2.2 워크스페이스 스위처
- 상단 워크스페이스 이름 클릭 → 드롭다운
- 소속 워크스페이스 목록: 아이콘 + 이름 + 멤버 수 + 내 역할
- 활성 워크스페이스: 체크마크(✓)
- "새 워크스페이스" 생성 옵션 (이름 입력 → 생성)
- "로그아웃" 버튼 (하단)
- 워크스페이스 전환 시 URL 변경 + 사이드바 새로고침

### 2.3 최근 방문 페이지
- 사이드바 상단 "최근" 섹션 (즐겨찾기 위, 접기 가능)
- 마지막 접근 5~10개 페이지
- localStorage 기반 (페이지 방문 시 자동 기록)
- 아이콘 + 제목, 전체 경로 툴팁
- "최근" 섹션 닫기 가능

### 2.4 사이드바 검색 필터
- Cmd+K 커맨드 팔레트 내 필터 시스템:
  - **필터 칩**: 생성자 (멤버 선택), 생성일 범위, 수정일 범위, 타입 (일반 페이지 / 데이터베이스)
  - **정렬**: 관련도순 (기본), 최신순, 이름순
  - **검색 범위**: 제목 + 블록 내용 (SP-8 검색 라우터 활용)
- 필터 상태 유지 (팔레트 닫았다 다시 열어도)

### 2.5 페이지 호버 프리뷰 (Peek)
- 사이드바 페이지 위 500ms 호버 → 프리뷰 카드
- 카드 내용: 커버 이미지(있으면), 아이콘 + 제목, 첫 3줄 텍스트, 마지막 편집일 + 편집자
- 카드 위치: 사이드바 오른쪽에 플로팅
- 마우스가 카드로 이동하면 유지, 벗어나면 300ms 후 사라짐

### 2.6 사이드바 접힘 시 호버 확장
- 접힌 상태에서 좌측 가장자리 호버 → 사이드바 오버레이로 임시 확장
- 메인 콘텐츠 밀리지 않음 (position: absolute 오버레이)
- 마우스 벗어나면 500ms 후 다시 접힘
- 사이드바 내에서 클릭하면 완전히 펼침 (고정)
- 펼침 상태에서 바깥 클릭 → 다시 접힘

### 2.7 키보드 페이지 트리 탐색
- **↑↓**: 페이지 아이템 간 이동 (포커스 링 표시)
- **Enter**: 선택된 페이지 열기
- **→**: 하위 펼치기, **←**: 접기 또는 상위 이동
- **Cmd+↑/↓**: 페이지 순서 위/아래 이동
- **Delete/Backspace**: 휴지통으로 이동 (Undo 토스트)
- **F2**: 페이지 이름 인라인 편집 (사이드바에서 바로)
- **Cmd+N**: 새 페이지 (현재 선택된 위치 기준)
- **Space**: 페이지 선택 토글 (멀티 선택)

### 2.8 브레드크럼 자식 드롭다운
- 브레드크럼 각 아이템 옆 ▼ 클릭 → 하위 페이지 드롭다운
- 하위 페이지 목록 (아이콘 + 제목)
- "새 하위 페이지" 생성 옵션
- 드롭다운 내 검색

### 2.9 페이지 이동 다이얼로그
- 블록 메뉴/더보기 메뉴에서 "이동" 옵션
- 모달: 워크스페이스 내 전체 페이지 트리 표시
- 대상 페이지 선택 → 하위 페이지로 이동
- 상단 검색으로 빠른 대상 찾기
- "최상위로 이동" 옵션

### 2.10 사이드바 섹션 접기
- "즐겨찾기", "최근", "개인 페이지" 섹션 헤더 클릭 → 접기/펼치기
- 접기 상태 localStorage 저장
- 접힌 섹션: 헤더만 표시 + 우측에 페이지 수 배지

### 2.11 페이지 아이콘 인라인 변경
- 사이드바 페이지 아이콘 클릭 → 이모지 피커 팝업 (페이지로 이동하지 않고)
- 빠른 아이콘 변경

### 2.12 탑바 더보기(···) 메뉴
- 현재 페이지 관련 모든 액션을 하나의 더보기 메뉴에:
  - 🔒 페이지 잠금/해제
  - ↔ 전체 너비 토글
  - ⭐ 즐겨찾기 추가/제거
  - 📋 복제
  - 📁 이동 (2.9 다이얼로그)
  - 📤 내보내기 → 서브메뉴: Markdown, HTML, PDF
  - 📥 가져오기 → 서브메뉴: Markdown, CSV
  - 🗑 삭제 (휴지통)
  - 🕐 히스토리 보기 (6.4)
  - 📊 페이지 통계 (6.11)
  - 📝 단어 수: 1,234 / 읽기 시간: 5분
  - 👤 마지막 편집: 홍길동, 3분 전

### 2.13 퀵 노트 (플로팅 버튼)
- 화면 우하단 플로팅 + 버튼
- 클릭 → 빠른 메모 입력 팝업 (미니 에디터)
- 저장 시 지정된 페이지(기본: "빠른 메모")에 블록 추가
- 또는 새 페이지로 생성

### 2.14 탭 / 분할 뷰
- 여러 페이지를 탭으로 열기 (브라우저 탭이 아닌 앱 내 탭)
- Cmd+클릭: 새 탭에서 열기
- 탭 드래그로 순서 변경
- 분할 뷰: 좌우 2개 페이지 동시 표시

### 2.15 뒤로/앞으로 네비게이션 히스토리
- Alt+← / Alt+→: 이전/다음 페이지 (브라우저 히스토리와 별도)
- 탑바에 ← → 버튼
- 방문 히스토리 스택 관리

## 3. 데이터베이스 UX (18개)

### 3.1 셀 인라인 편집 개선
- **싱글 클릭** → 즉시 편집 모드 (현재: 추가 클릭 필요)
- **Tab** → 다음 셀, **Shift+Tab** → 이전 셀
- **Enter** → 다음 행 같은 열, **Shift+Enter** → 셀 내 줄바꿈
- **Escape** → 편집 취소
- **텍스트 셀**: 클릭 즉시 커서 배치, 더블클릭으로 단어 선택
- **Select 셀**: 클릭 즉시 드롭다운 열림, 타이핑으로 검색
- **날짜 셀**: 클릭 즉시 날짜 피커 열림
- **체크박스 셀**: 클릭 즉시 토글 (편집 모드 없이)
- **방향키**: 셀 간 이동 (편집 모드가 아닐 때)

### 3.2 행 드래그 순서 변경
- **테이블**: 행 좌측 핸들(⋮⋮) 드래그
- **보드**: 카드 칼럼 내 순서 + 칼럼 간 이동
- **리스트**: 행 핸들 드래그
- **드롭 인디케이터**: 수평선(행 사이)
- 순서 변경 시 `database.updateRow` position 업데이트

### 3.3 그룹 접기/펼치기
- 그룹 헤더 클릭 → 접기/펼치기
- 접힌 상태: 헤더만 + 행 수 카운트
- "모두 접기" / "모두 펼치기" 버튼 (도구 모음)
- 접기 상태 로컬 저장 (viewId + groupValue 키)

### 3.4 뷰 탭 드래그 순서
- 뷰 탭 드래그로 순서 변경
- 뷰 탭 우클릭 → 복제, 삭제, 이름 변경
- 더블클릭: 이름 인라인 편집

### 3.5 속성 열 드래그 순서
- 테이블 뷰 컬럼 헤더 드래그로 순서 변경
- 드롭 인디케이터 (파란 수직선 2px)
- title 열은 항상 첫 번째 (이동 불가)

### 3.6 행 높이 설정
- 뷰 설정: 행 높이 옵션
  - **기본** (1줄, 36px)
  - **중간** (2줄, 56px)
  - **큰** (3줄, 76px)
  - **자동** (내용에 맞춤)
- 뷰별 개별 설정

### 3.7 데이터베이스 행 피크 뷰 (사이드 패널)
- 행 클릭 → 오른쪽 사이드 패널로 열기 (현재: 전체 페이지 이동)
- 패널: 속성 편집 + 블록 에디터 (축소판)
- "전체 페이지로 열기" 버튼
- ESC / 패널 바깥 클릭으로 닫기
- 패널 너비 드래그 조절 (기본 400px, 최소 300px, 최대 600px)

### 3.8 캘린더 드래그로 날짜 변경
- 캘린더 카드를 다른 날짜 셀로 드래그 → date 속성 자동 변경
- 드래그 중 대상 날짜 하이라이트 (파란 배경)
- 멀티데이 이벤트: 양쪽 끝 드래그로 기간 변경

### 3.9 타임라인 드래그로 기간 변경
- 바 양쪽 끝 드래그 → 시작/종료 날짜 변경
- 바 전체 드래그 → 날짜 이동 (기간 유지)
- 드래그 중 날짜 툴팁 표시
- 스냅: 일 단위로 정렬

### 3.10 링크드 데이터베이스
- 기존 DB를 다른 페이지에서 "링크드 뷰"로 참조
- 원본 DB 데이터를 독립적인 필터/정렬/뷰로 표시
- 슬래시 커맨드: "링크드 데이터베이스" → DB 검색/선택
- 원본 DB 속성 변경 시 링크드 뷰에도 반영

### 3.11 행 템플릿
- 새 행 추가 시 사용할 템플릿 정의
- 기본 속성 값 + 블록 구조 사전 설정
- "새 행" 버튼 옆 ▼ → 템플릿 목록 + "새 템플릿 만들기"
- 템플릿 편집/삭제

### 3.12 CSV 가져오기
- DB 도구 메뉴에서 "CSV 가져오기"
- 파일 업로드 → 미리보기 (첫 10행)
- 첫 행 = 속성 이름, 나머지 = 행 데이터
- 속성 타입 자동 감지 (숫자, 날짜, URL, 텍스트)
- "가져오기" 확인 → 일괄 행 생성

### 3.13 데이터베이스 잠금
- **구조 잠금**: 속성/뷰 변경 불가 (행 데이터는 편집 가능)
- **완전 잠금**: 모든 변경 불가
- 잠금 아이콘 + "잠김" 배지
- OWNER/ADMIN만 잠금 해제 가능

### 3.14 수식 속성 기본 지원
- 수식 편집기: 텍스트 입력 + 자동완성
- 기본 함수:
  - 속성 참조: `prop("속성이름")`
  - 조건: `if(조건, 참, 거짓)`, `empty(값)`
  - 수학: `add`, `subtract`, `multiply`, `divide`, `mod`, `pow`, `abs`, `round`, `ceil`, `floor`
  - 집계: `sum`, `average`, `min`, `max`, `count`
  - 문자열: `length`, `contains`, `replace`, `slice`, `lower`, `upper`, `trim`
  - 날짜: `now()`, `dateAdd`, `dateBetween`, `formatDate`, `year`, `month`, `day`
  - 논리: `and`, `or`, `not`
- 문법 검사: 실시간 에러 표시 (빨간 밑줄 + 에러 메시지)
- 자동완성: 속성 이름, 함수 이름 제안

### 3.15 Relation & Rollup 기능 구현
- **Relation**: 다른 DB의 행을 참조하는 속성
  - 셀 클릭 → 대상 DB 행 검색/선택 팝업
  - 다대다 관계 지원
  - 양방향 relation 자동 생성
- **Rollup**: relation으로 연결된 DB의 값을 집계
  - 설정: relation 속성 선택 → 대상 속성 선택 → 집계 함수 (count, sum, average, min, max, show original)

### 3.16 데이터베이스 하단 합계 행
- 테이블 뷰 하단에 합계 행 표시
- 각 숫자 속성: 합계, 평균, 최소, 최대, 카운트 중 선택
- 클릭하여 집계 함수 변경

### 3.17 셀 내 멀티라인 표시
- 텍스트/제목 셀: 줄바꿈 허용 (Shift+Enter)
- 행 높이 "자동"일 때 셀 높이 확장
- 텍스트 잘림: 고정 높이 시 말줄임표(...) 표시

### 3.18 속성 헤더 클릭 → 정렬
- 테이블 뷰 컬럼 헤더 클릭 → 오름차순 정렬
- 다시 클릭 → 내림차순
- 다시 클릭 → 정렬 해제
- 정렬 중인 열에 ▲/▼ 아이콘 표시

## 4. 협업 & 소통 UX (12개)

## 5. 비주얼 폴리시 & 모션 (14개)

### 5.1 페이지 전환 애니메이션
- 콘텐츠 fade-in: opacity 0→1, 150ms ease-out
- 약간 위로 슬라이드: translateY(8px)→0
- loading.tsx로 스켈레톤 표시 (5.8)

### 5.2 사이드바 모션 개선
- 접기/펼치기: cubic-bezier(0.4, 0, 0.2, 1), 300ms
- 페이지 트리 확장/축소: max-height + opacity 전환 (200ms)
- 드래그 리사이즈: transform으로 실시간 (layout thrash 방지)
- 호버 확장(2.6): translateX 전환

### 5.3 토스트 스택 애니메이션
- 입장: translateY(100%)→0 + opacity 0→1, 300ms spring
- 퇴장: translateY(0)→100% + opacity 1→0, 200ms ease-in
- 스택: 새 토스트 → 기존 토스트들 위로 자동 이동 (40px gap)
- 최대 3개 동시, 초과 시 가장 오래된 것 제거

### 5.4 모달/다이얼로그 트랜지션
- 배경: opacity 0→0.6 (200ms)
- 모달: scale(0.96)→1 + opacity 0→1 (200ms cubic-bezier)
- 닫힘: 역순 (150ms)
- 모든 모달 컴포넌트에 일관 적용

### 5.5 드롭다운/팝오버 애니메이션
- 열림: scaleY(0.96)→1 + opacity 0→1, origin top, 150ms
- 닫힘: 즉시 (150ms 이내)
- 적용 대상: 슬래시 메뉴, 블록 메뉴, 컬러 피커, AI 메뉴, 멘션 팝업

### 5.6 블록 삽입/삭제 애니메이션
- 삽입: height 0→auto + opacity 0→1, 200ms ease-out
- 삭제: height auto→0 + opacity 1→0, 150ms ease-in
- 이동: FLIP 애니메이션 (First→Last→Invert→Play)

### 5.7 셀렉션 애니메이션
- 블록 선택 하이라이트: background-color 전환 150ms
- 텍스트 선택 시 인라인 툴바: opacity 0→1 + translateY(4px→0), 150ms

### 5.8 로딩 스켈레톤
- **페이지 로딩**: 제목 스켈레톤 (넓은 바) + 본문 스켈레톤 (3~5줄 바, 너비 랜덤)
- **사이드바**: 페이지 아이템 스켈레톤 (아이콘 원 + 텍스트 바)
- **데이터베이스**: 테이블 스켈레톤 (헤더 + 5행 바)
- 펄스 애니메이션: background-position 이동 (shimmer 효과)

### 5.9 에러 바운더리
- **에디터 에러**: "에디터를 불러오지 못했습니다" + "새로고침" 버튼
- **사이드바 에러**: "페이지 목록을 불러오지 못했습니다" + "다시 시도"
- **데이터베이스 에러**: "데이터를 불러오지 못했습니다" + "다시 시도"
- React ErrorBoundary 컴포넌트로 각 영역 감싸기

### 5.10 스크롤 관련
- `scroll-behavior: smooth` 전역 적용
- **맨 위로 버튼**: 우하단 플로팅, 500px 이상 스크롤 시 나타남 (fade-in)
- **목차 클릭**: 해당 헤딩으로 부드러운 스크롤
- **스크롤바 스타일**: 노션 스타일 얇은 스크롤바 (이미 CSS에 있음, 확인)

### 5.11 커서 상태 피드백
- 에디터: text 커서
- 드래그 핸들: grab → grabbing
- 리사이즈 핸들: ew-resize, ns-resize, nwse-resize, nesw-resize
- 링크: pointer
- 비활성 영역: default
- 로딩 중: wait / progress

### 5.12 호버 상태 일관성
- 모든 클릭 가능 요소: hover 시 배경색 변경 (var(--bg-hover))
- 모든 버튼: hover 시 약간 밝아짐 또는 그림자
- 사이드바/에디터/데이터베이스 전체 호버 스타일 감사 및 통일

### 5.13 포커스 링 (키보드 접근성)
- Tab으로 이동한 요소: 파란 포커스 링 (outline: 2px solid #2383e2)
- 마우스 클릭 시: 포커스 링 숨김 (:focus-visible만 표시)
- 모든 인터랙티브 요소에 적용

### 5.14 다크모드 전환 트랜지션
- 테마 변경 시 전역 CSS 트랜지션: `transition: background-color 300ms, color 300ms, border-color 300ms`
- 부드러운 색상 전환 (깜빡임 없이)
- html에 `[data-transitioning]` 클래스 임시 추가 → 전환 후 제거

- + 버튼 호버 1초: "클릭해서 아래에 블록 추가" 툴팁
- ⋮⋮ 버튼 호버 1초: "드래그해서 이동, 클릭해서 메뉴 열기" 툴팁
- 노션 스타일: 작은 회색 말풍선

### 7.4 빈 페이지 안내 UI
- 새로 만든 빈 페이지:
  - 타이핑 placeholder: "'/'를 입력하여 명령어를 사용하거나 입력을 시작하세요"
  - 하단에 빠른 블록 버튼 (fade 표시):
    - 📝 텍스트, 📋 할 일, 📊 표, 🖼 이미지, 📄 템플릿에서 시작
  - 버튼 클릭 시 해당 블록 삽입

### 7.5 커서 스타일 피드백
- 에디터 텍스트: text (|)
- 드래그 핸들 ⋮⋮: grab, 드래그 중: grabbing
- 이미지 리사이즈: nwse-resize, nesw-resize, ew-resize, ns-resize
- 사이드바 리사이저: col-resize
- 링크: pointer
- 비활성(잠긴 에디터): not-allowed

### 7.6 Cmd+Enter 빠른 블록 삽입
- 어디에서든 Cmd+Enter → 현재 블록 아래에 새 빈 paragraph + 포커스
- 토글 안에서: 토글 내부에 삽입
- 리스트 안에서: 같은 리스트 레벨에 삽입

### 7.7 Shift+Enter 소프트 브레이크
- 블록 내 줄바꿈 (새 블록 생성 X)
- ProseMirror의 `hardBreak` 노드
- 리스트 아이템 내에서도 동작

### 7.8 Backspace 블록 병합
- 블록 시작에서 Backspace → 이전 블록과 텍스트 병합
- 빈 블록에서 Backspace → 블록 삭제 + 이전 블록 끝으로 커서
- 리스트 아이템에서 Backspace → 리스트 해제 (일반 paragraph로)
- 첫 블록 시작에서 Backspace → 무시

### 7.9 Enter 블록 분할
- 블록 중간에서 Enter → 커서 위치에서 블록 분할 (위: 커서 전, 아래: 커서 후)
- 리스트에서 Enter → 새 리스트 아이템
- 빈 리스트 아이템에서 Enter → 리스트 해제 + 일반 paragraph
- 제목 끝에서 Enter → 아래에 일반 paragraph (제목이 아님)

### 7.10 ESC 동작 계층
- ESC 1회: 현재 메뉴/팝오버/드롭다운 닫기
- ESC 2회: 블록 선택 해제
- ESC 3회: 에디터 포커스 해제
- 이미 최상위: 아무 동작 없음
- 각 단계에서 시각적 피드백 (포커스 링 변경)

### 7.11 Cmd+Shift+K 링크 제거
- 커서가 링크 위에 있을 때 Cmd+Shift+K → 링크 해제 (텍스트 유지)
- 또는 인라인 툴바 링크 버튼 다시 클릭

### 7.12 Cmd+Backspace 블록 내 전체 텍스트 삭제
- 블록 내 커서 위치부터 블록 시작까지 삭제
- 블록 자체는 유지 (빈 블록)

## 8. 성능 최적화 (6개)

### 8.1 React.memo 최적화
- SidebarPageItem: 페이지 데이터가 변경되지 않으면 리렌더링 방지
- CellRenderer: 셀 값이 변경되지 않으면 리렌더링 방지
- 데이터베이스 행/열 컴포넌트: memo 적용

### 8.2 tRPC 쿼리 최적화
- `staleTime: 30000` (30초) 설정으로 불필요한 재요청 방지
- `refetchOnWindowFocus: false` (탭 전환 시 재요청 방지)
- 사이드바 페이지 리스트: `staleTime: 60000`

### 8.3 이미지 레이지 로딩
- 에디터 내 이미지: `loading="lazy"` 속성
- 뷰포트 밖 이미지는 로드하지 않음
- Intersection Observer로 가시성 감지

### 8.4 번들 사이즈 최적화
- 코드 스플리팅: 데이터베이스 뷰 컴포넌트 dynamic import
- highlight.js: 전체 대신 필요한 언어만 import
- date-fns: tree-shaking으로 필요한 함수만

### 8.5 에디터 디바운스 최적화
- onUpdate 디바운스: 현재 1초 → 유지 (적절)
- 검색 디바운스: 현재 300ms → 유지
- 제목 저장 디바운스: 현재 500ms → 유지
- tRPC 캐시 무효화: 배치 처리 (여러 mutation 후 한 번만)

### 8.6 데이터베이스 쿼리 최적화
- 속성 수 많은 DB: 필요한 속성만 select
- 행 수 많은 DB: 서버 사이드 페이지네이션 (cursor 기반)
- 뷰 전환 시: 이전 뷰 데이터 캐시 유지
